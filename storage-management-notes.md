# Storage Management Notes

## Objective
Resolve low storage issue on the C drive by reallocating unused disk space.

## Problem
- C drive was almost full.
- D drive contained a large amount of unused storage.
- Insufficient storage could impact software installation and system performance.

## Tools Used
- Disk Management
- File Explorer

## Shortcuts Used
- Win + R → Open Run Dialog
- diskmgmt.msc → Open Disk Management
- Win + E → Open File Explorer
- Win + I → Open Settings
- Ctrl + Shift + Esc → Open Task Manager

## Process
1. Analyzed disk partitions.
2. Identified partitions preventing C drive expansion.
3. Removed the unused partition.
4. Created unallocated space.
5. Extended the C drive.
6. Verified the new storage allocation.

## Concepts Learned
- SSD
- Disk Partitions
- Unallocated Space
- Volume Extension
- Storage Management

## Outcome
Successfully increased available storage on the C drive and resolved the storage issue.
